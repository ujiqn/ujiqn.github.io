Apple ファミリー共有について

最終更新日: 2026年6月24日

Apple ファミリー共有は、家族の Apple ID をひとつのグループにまとめる Apple の標準機能です。

お子さまの Apple ID をご家族のグループに加えていただくことで、お子さまが自分でミマリモを削除したり、iCloud からサインアウトしてしまうことを防げます。

ご利用にあたっては、お子様の Apple ID の年齢が 17 歳以下に設定されている必要があります。

なお、上記の保護機能（アプリ削除の防止など）は iOS 26 以降で動作確認済みです。iOS 17〜25 では、ファミリー共有との連携自体は成立しますが、保護機能が動作しない場合があります。お子さまの端末を最新の iOS にアップデートしておくと、より安心してご利用いただけます。

設定の流れ

おおまかな手順は以下のとおりです。

  1. 親のデバイスで「設定」を開く
  2. 一番上の[ご自身のお名前]→「ファミリー共有」 を開く
  3. 「メンバーを追加」からお子さまの Apple ID を招待します。お子さまがまだ Apple ID をお持ちでない場合は、ここで新規作成できます

詳しい手順や、お子様の Apple ID の作り方は Apple のサポートページをご確認ください。

  • ファミリー共有を設定する（Apple サポート）
  • お子様の Apple ID を作成する（Apple サポート）
