接続を解除したいけれど、親のデバイスが手元にありません

ミマリモは、親のデバイスから「接続を解除」するのが基本です。親のデバイスがすぐに使えないときは、子のデバイスから次の方法で解除できます。

かんたんモードでつないでいるとき

子のデバイスで iOS の「設定」アプリを開き、次の順番でタップしてください。

  1. 「スクリーンタイム」
  2. 一番下までスクロールして「スクリーンタイムをオフにする」

これで、ミマリモのロックは解けます（端末のパスコードは必要ですが、スクリーンタイム用のパスコードは不要です）。

ファミリー共有モードでつないでいるとき

このモードでは、子のデバイスからは解除できません。親が新しいデバイスから次の手順で解除してください。

  1. 親の Apple ID で iCloud にサインイン
  2. 「設定」→「スクリーンタイム」→「スクリーンタイム・パスコード」を再設定（パスコードを忘れた場合は「Apple ID でリセット」）
  3. 「コンテンツとプライバシーの制限」をオフにする

設定はネットワーク経由で子のデバイスに同期され、ロックが解けます。子のデバイスを直接触る必要はありません。

親のデバイスで電源ボタンを押しても、子のデバイスで反映されません

まず、子のデバイスで iOS の「設定」→「スクリーンタイム」→「ミマリモ」が ON になっているかをご確認ください。OFF になっていると、ミマリモはロックをかけたり解いたりすることができません。

子のデバイスでスクリーンタイムを ON/OFF した直後は、その変化がミマリモのアプリにすぐ伝わらないことがあります（iOS の仕組みによるものです）。お手数ですが、子のデバイスでミマリモを一度完全に終了し、もう一度開き直してください。最新の状態が反映されます。

完全に終了する手順：

  1. ホーム画面に戻る
  2. 画面の一番下からゆっくり上にスワイプして、画面の真ん中で指を止める
  3. アプリのカードが並んだ画面が出るので、ミマリモのカードを上にスワイプして消す
  4. もう一度ミマリモのアプリを開く

それでも反映されないときは、お問い合わせ窓口までご連絡ください。

親のデバイスを買い替えました。どうすればつなぎ直せますか？

  1. 新しい親のデバイスにミマリモをインストールしてください
  2. 設定の「接続中のデバイス」から、古いデバイスとの接続を解除します
  3. 新しいコードを発行して、子のデバイスに入力します

これで、新しい親のデバイスから子を操作できるようになります。買い切り購入は Apple ID に紐づくため、新しいデバイスでも引き継がれます（Apple のファミリー共有を使っている場合は、家族の他の親のデバイスにも自動的に共有されます）。

ファミリー共有と連携してもうまく機能しない場合

ファミリー共有による強い保護（アプリの削除防止など）は、お子さまの端末が iOS 26 以降で動作確認済みです。iOS 17〜25 では、ファミリー共有との連携自体は成立しますが、保護機能が動作しない場合があります。

お子さまの端末を最新の iOS にアップデートしておくと、より安心してご利用いただけます。

また、ファミリー共有のご利用には以下の条件があります。

  • 親と子それぞれの Apple ID で「ファミリー共有」を有効にする
  • お子さまの Apple ID をご家族のグループに加え、「お子様アカウント」として設定する
  • 親の端末でスクリーンタイムのパスコードを設定する（お子さまの端末のパスコードとは別の 4 桁）

これらが整っている場合、お子さまの端末でミマリモを開くと、自動的にファミリー共有との連携が成立します（手動操作は不要です）。

以前使っていた iPhone/iPad をお子さま用に使う場合の準備

ご家族の使い古しの iPhone や iPad をお子さま用に転用する場合は、必ず端末をいったん初期化（リセット）してから、お子さまの Apple ID でセットアップしてください。

「設定」から自分の Apple ID をサインアウトしてお子さまの Apple ID でサインインし直すだけでは、以前のアカウントの設定情報が端末に残り、ファミリー共有や Mimarimo の連携が正しく動かないことがあります。

初期化の手順：

  1. （任意）必要なデータをバックアップ
  2. 「設定」→「一般」→「転送または iPhone をリセット」
  3. 「すべてのコンテンツと設定を消去」をタップ
  4. デバイスパスコードと Apple ID パスワードを入力
  5. リセット完了後、初期セットアップで お子さまの Apple ID でサインイン
  6. ファミリー共有の招待を承認

その後、ミマリモをインストールしてご利用ください。

参考: Apple Support — iPhone を売却・譲渡・下取りに出す前にやること

さいしょからやり直したいです

子のデバイスからミマリモを削除して、もう一度インストールしてください。これで最初の状態（接続なし・認可なし）に戻ります。そのあと、親のデバイスから新しいコードを発行して、もう一度つなぎ直してください。

ブロック中でも電話やメッセージは使えますか？

はい。電話・メッセージなど iOS の標準アプリは、ミマリモのロックの対象外です。緊急の連絡は妨げません。

ただし、これは iOS の仕組みによるものであり、緊急時に必ず通じることを保証するものではありません。

ミマリモは何歳のお子さん向けですか？

ミマリモは「まだ自己管理が難しい年代に、親が寄り添うリモコン」として設計しています。具体的には未就学児（3〜6 歳）から小学生くらいまでが想定の中心です。

中学生以上のお子さまにもミマリモは技術的にはお使いいただけますが、おすすめしていません。

分別のつく年代に本人の意思に反してブロックを使うと、信頼関係が崩れたり、抜け道探し・反発を招きやすいためです。本来この年代に必要なのは「親が止める」ことではなく、「約束を守る」「自分で律する」ことを身につける段階への移行だと考えています。

加えて、お子さまが Apple ID やパスコードの仕組みを理解できる年齢になると、技術的にも回避経路が出てきます（親のデバイスを借りる／パスコードを聞き出す等）。これは弱点ではなく、ミマリモが強制ではなく約束のためのリモコンとして作られていることを反映しています。

成長されたら、ミマリモを卒業して使うのをやめていただいて構いません。買い切りなので、解約手続きは不要です（アプリを削除いただくだけで OK です）。

ミマリモはペアレンタルコントロールアプリですか？

「約束を守るためのリモコン」と呼んでいます。

iOS の「スクリーンタイム」と同じ仕組み（Family Controls）を使っていますが、目的は監視や制限の強化ではなく、親子の約束を端末上で見えるようにすることです。位置情報、Web 閲覧履歴、SNS の中身、利用時間の集計といった機能は持っていません（親の操作で「いま使える／いまおやすみ」を切り替えるだけ）。
